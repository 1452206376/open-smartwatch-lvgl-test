# 使用方法：
`git clone https://github.com/1452206376/open-smartwatch-lvgl-test.git`   
把libraries文件夹拖到 `C:\Users\用户名\文档\Arduino\`  
如果之前安装过lvgl要删除，如果不想删除要手动配置`lv_ex_conf.h`并确认lvgl版本为7.9.0  
然后开发板选择`ESP32 Dev Module`，上传
